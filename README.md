# Netflix Stock Price — Exploratory Data Analysis

## Overview
An end-to-end EDA project on Netflix stock price data spanning 
25 years (2002–2026) covering 5,959 trading days.

## Dataset
- Source: Public Netflix stock price data
- Rows: 5,959 | Columns: 6
- Features: Date, Open, High, Low, Close, Volume

## Key Findings
1. Netflix stock grew from under $1 in 2002 to over $900 by 2026
2. 2022 saw the biggest crash — stock dropped over 70% in one year
3. Trading volume was highest during 2020-2022 (COVID + crash period)
4. Average daily price range increased significantly after 2015

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Project Structure
- Netflix_stock_EDA_project.ipynb — main analysis notebook
